# 1.0.1

### 重点更新

- 新增 iPhone 录制支持。
- 新增键盘录制支持。
- 新增 Live Photo 导出支持。
- 更新录制工具栏，整体展示更清晰、更精致。
- 重做导出面板，现在以编辑器内浮层展示，尺寸变化更顺滑，控件更清晰，并支持 Live Photo 预览、切换与 60 秒裁剪。
- 时间线预览新增播放速度菜单，支持 0.25x 到 2x，仅影响预览。
- 设置中新增「开机启动」选项，并优化通用设置。
- 新增独立「外观」设置页，可调整主题色、工具栏材质和提示材质。
- 颜色选择器支持输入 Hex 颜色值。
- 录制控件、检查器标签和撤销/重做快捷键新增提示。
- 区域选择支持输入尺寸、中心参考线、比例预设，以及 Shift / Option 缩放快捷操作。
- 摄像头美颜支持录制中预览和开关，可设为录制默认值，也可在编辑器中继续调整，并支持基于人脸的形变控制。
- 优化默认摄像头动画参数，使用更柔和的弹簧曲线和更明显的运动模糊，让缩放动画更自然。
- 快捷键设置优化为更大的动态键帽，并支持移除绑定。
- 编辑器顶部新增更多导出画布比例预设、源画面裁剪，以及最高 5K 的源尺寸导出。
- 新录制会先自动保存为 .cam 工作区，再进入编辑器；默认保存位置可配置。
- 录制设置中新增工作区和原始录制缓存大小展示，并支持清理旧缓存或全部缓存。
- 编辑器标题现在可以在 Finder 中显示当前工作区。
- 关于页面支持导出最近错误日志，方便排查问题。
- 停止录制后会立即进入 Processing 状态，并延长最终处理保活时间，提高稳定性。

### 问题修复

- 修复源尺寸导出未正确匹配最终预览合成的问题。
- 修复导出画布因像素取整出现黑边的问题。
- 修复显示器和笔记本模型导出中的动态壁纸、边框背景和圆角裁剪问题。
- 修复全屏切换时窗口红黄绿按钮位置异常的问题。
- 修复窗口录制默认圆角，现在会根据捕获到的窗口遮罩计算，而不是使用固定估算值。
- 视口圆角和 CLI 窗口捕获圆角上限提升至 120。
- 修复满音量导出时音频混流导致文件体积异常增大的问题。
- 修复模型缩放靠近源画面边缘时的吸附异常。
- 修复 Intel Mac 上捕获目标选择和菜单栏更新可能崩溃的问题。
- 修复多显示器环境下，副屏窗口和区域选择不可用的问题。
- 修复区域选择窗口层级、取消按钮点击范围，以及开始按钮悬停崩溃问题。
- 修复 MP4 导出在 Preview 和 Finder 中的兼容性问题。
- 修复预览摄像头动画，使边缘缩放路径更平滑，自动跟随更稳定，方向运动模糊更准确。
- 修复颜色选择器拖拽点在调整颜色时位置漂移的问题。
- 修复拒绝屏幕录制权限后仍显示错误的问题。
- 修复重复录制片段导出时复用旧帧的问题。
- 修复高分辨率录制下 4K 导出卡在 3% 左右的问题。
- 修复预览播放和导出时画面变空或被悬停拖动影响的问题。
- 修复录制过程中切换桌面后工具栏消失的问题。
- 优化录制工具栏音频表显示，音量变化更明显。
- 修复开启系统音频后录制处理卡住的问题。
- 修复未保存编辑器关闭或退出时卡在系统保存面板的问题，现在会使用 ScreenCam 自己的提示。
- 修复带动态壁纸或摄像头图层的长视频导出卡在 79% 左右的问题。

### 性能优化

- 优化包含大量键盘事件时的编辑器缩放性能。
- 优化录制流程，使用更轻量的视频缓冲。
- 减少窗口录制并开启系统音频时的额外捕获开销。
- 减少预览播放时不必要的鼠标追踪。
- 优化长时间高分辨率导出，使用更快的分块渲染，降低 CPU 和内存占用。
