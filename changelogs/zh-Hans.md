# 1.0.5

- 导出可在后台继续执行，并显示剩余时间预估、可自定义材质的刘海进度、完成后的操作按钮和窗口 Genie 恢复动画。
- 时间线剪辑提供独立剪辑模式，支持按住 Option、剪刀光标和录制音频联动聚焦。
- 光标设置可在静止时自动隐藏光标，并支持可下载点击音效，预览和导出都会播放，已下载音效可离线使用。
- mockup 素材可从素材目录下载和更新，系列、型号、颜色以及 Apple Watch 表壳和表带选择会立即反映到预览。
- GIF 导出新增 1080p 及以下分辨率档位，并会在导出前显示预计文件大小。
- 已有工程会在迁移素材下载完成前保留已选择的 mockup。
- GIF 导出能更准确保留浅色界面颜色和特效图层几何。
- 在 GIF 和 Live Photo 导出之间切换时，裁剪预览保持稳定，GIF 格式选择器也可在整个分段区域内响应点击。
- 很短的时间线片段更容易选中，不会因单击误触裁剪。
- mockup 素材目录下载占用更少内存，长 GIF 导出也会减少帧缓冲分配。

- 文字效果框缩小时会保留设置的字号。

- 裁短主录制片段后可以再拖回原始长度，编辑器窗口较矮时密集时间线也可以纵向滚动。
- 工具栏的打开工程会从默认录制工程文件夹开始。

- 相机动画默认参数使用更柔和的弹簧曲线。

- 时间线放大并横向滚动时，比例尺、播放刻度、悬停预览和片段光标保持对齐，缩小后左侧会回到 00:00。
- 关闭或删除系统音频片段或轨道后，编辑器预览会完全静音系统声音。
- 密集时间线下的特效预览、变焦播放、导出和大型工程编辑更快。

# 1.0.2

- 背景为无时，屏幕尺寸会提示导出黑边并可一键移除。
- 固定屏幕尺寸在锁定比例时可以正常缩小，高度拖拽方向也更自然。
- Zoom clip 缩放等级上限放宽到 5x，手动变焦进出目标点时也更稳定。
- 录制中的相机预览会提示仅供预览，并可选择以后不再显示。

- 录制相机预览和编辑器预览播放开启美颜时会降低 CPU 占用。
- 复杂合成渲染预览时，不再阻塞编辑器关闭或系统文件面板。
- 调整相机美颜时会避免预览 CPU 异常飙升。
- 相机美颜现在支持自然的局部唇色、更稳定的补光和磨皮。
- 自动保存的 .cam 工程会保留文件夹访问权限，在受保护位置保存和关闭编辑器更可靠。
- 相机美颜导出现在更快，尤其是小相机画面和较低分辨率输出。
- 比例菜单会为每个预设显示形状预览。
- 保存编辑器预设不再导致预设菜单崩溃。
- 设备外观会在需要时缩小屏幕开口，让录制内容铺满屏幕且不留黑边。
- 付费页购买会避免把 Apple 确认框绑定到特定应用窗口。
- 多显示器场景下 Apple 订阅确认框更可靠，并会针对疑似确认异常记录聚焦诊断。

- 录制设置会显示自动保存工程和原始录制缓存大小，并支持清理过期或全部缓存项目。
- 设置新增外观控制，可调整主题色、工具条材质和提示材质。
- 设置新增开机自动启动、录制时放大的快捷键按键、快捷键删除和最近错误日志导出。
- File 菜单新增懒加载的最近打开和默认目录子菜单，可快速打开已保存工程。
- 编辑器顶部栏支持保存、重命名、排序、删除、确认应用、分享和导入预设。
- 编辑器顶部栏新增更多导出画布比例、原画面裁切和最高 5K 的源尺寸导出。
- 编辑器标题可在 Finder 中显示工程文件。
- 时间线轨道排序会作为编辑器偏好保留，隐藏状态会在每次编辑会话重置。
- 时间线和原画面裁切支持触摸板捏合缩放，裁切弹窗缩放后还可以平移查看。
- 区域选择支持手动输入尺寸、居中参考线、比例预设和 Shift/Option 缩放快捷键。
- 时间线预览控制新增 0.25x 到 2x 的播放速度。
- 摄像头美颜可在录制时预览和切换，可保存为默认值，也可在编辑器中继续调整并使用人脸感知形变控制。
- 默认镜头动画使用更柔和的 spring 配置和更强的运动模糊，让缩放更平滑。
- 预览镜头运动会保持边缘约束放大和缩小路径为直线，并从正确缩放中心生成方向性运动模糊。
- 光标回放能更紧跟录制时的移动。
- 裁切原画面后，光标位置和相机焦点会保持对齐。
- 预览播放时摄像头画面会与相框动画保持同步。
- 结束录制后工具条会立即进入处理中状态，并延长收尾等待时间。
- 录制时切换桌面会保持浮动录制工具条可见。
- 录制工具条的音频电平会显示更清晰的起伏。
- 从浮动录制工具条打开区域录制时，区域选择层会稳定显示在前方。
- 多显示器环境下可以在副屏选择窗口和区域录制目标。
- 拒绝屏幕录制权限时会取消录制且不再显示错误。
- 捕获选择流程避免 Intel TestFlight 崩溃，包括目标悬停、菜单栏状态菜单更新和从 Dock 重新打开。
- 捕获选择确认栏的取消按钮现在整个圆形控件都可点击。
- 鼠标悬停在捕获选择开始按钮上时不再导致应用崩溃。
- 开启系统音频的录制可以正常结束，不再卡在处理中。
- 关闭或退出未保存的编辑器会进入 ScreenCam 关闭流程，不再触发系统文档保存面板。
- 工程自动保存会等待系统文件面板关闭，并更安全地写入 .cam 工程。
- 拖动时间线和被打断的交互不再导致预览播放卡住，或在暂停时跳到末尾。
- 调整或拖动时间线片段时，播放刻度不再在按下瞬间跳转。
- 预览音频的静音和音量控制会保持同步、平滑过渡，并且不再加入撤销步骤。
- 颜色选择器支持手动输入十六进制颜色，并在拖拽色板或色条时保持指示器位置正确。
- 检查器文案统一使用屏幕和设备概念，尺寸预设选择器也与其他菜单控件保持一致。
- 录制控件、检查器标签页和撤销/重做快捷键会显示提示。
- 清透提示气泡会保留轻微深色底色，在白色背景上也能看清。
- 导出的 MP4 使用更兼容 macOS 预览播放的 H.264 帧顺序，避免 quarantine 元数据，并保持正常速度音频时序。

- 导出画布不再因像素取整出现边缘黑边。
- display mockup 导出时，动态壁纸和背景会正确铺到屏幕框内。
- 笔记本设备导出会按屏幕圆角裁剪录制内容。
- mockup 缩放可以跟随鼠标越过录屏源边缘，并避免放大的设备外框正好贴住画布边缘。

- 录制、预览播放、高分辨率导出和大型键盘时间线会减少 CPU 与内存占用。
- 编辑器窗口全屏切换时会保持红黄绿窗口按钮的原生位置。
